# Web приложение для горнолыжного курорта
# Описание
Web-приложение для горнолыжного курорта — это система для автоматизации продажи и управления пропусками. Система позволяет централизованно оформлять, оплачивать и активировать электронные ski-pass, предоставляя гостям удобный цифровой сервис, а курорту — полный контроль над доступом на склоны.

Система закрывает ключевые потребности:
 - Централизованное управление всеми пропусками в одном месте
 - Автоматизация продажи и активации ski-pass онлайн
 - Исключение очередей в кассах и потерь времени для гостей
 - Контроль доступа на подъемники в режиме реального времени

Гостям курорта выгодно использовать наш продукт, потому что его можно легко адаптировать под конкретные нужды с помощью встроенного конфигуратора. С помощью конфигуратора можно настроить:

- Тарифы и типы пропусков (взрослый/детский, на 1 день/сезонный)
- Зоны доступа для разных категорий ski-pass
- Скидки и специальные предложения
